# `chain1d_mag` — magnetized 1D tight-binding chain (G2 prerequisite)

w2s-generated model for the **magnetic chain**

    H = -2 gamma cos(k a) ⊗ I_spin  -  J_ex sigma_z ,   gamma = 1, J_ex = 0.1

A 1-site, 2-orbital chain. **Spin convention (sup/sdw):** the spin channel is set by the
orbital LABEL — `_s+_` = spin↑ (s_z=+1), `_s-_` = spin↓ (s_z=-1). Orbital 1 = `A_s+_` (↑),
orbital 2 = `A_s-_` (↓); nearest-neighbour hop `-gamma` (diagonal in spin), on-site
exchange `diag(-J_ex, +J_ex)`. This is the model the G2 spin tests (#6 precession,
#7 flavor discrimination) need.

## Build & verification
```
wannier2sparse chain1d_mag <NX> 1 1 VX SX SY SZ -o operators
```
Verified (NX=64 ⇒ 128×128):
- **HAM**: Hermitian, exchange-split spectrum `[-2.1, 2.1]` (split `2·J_ex=0.2`).
- **Spin operators** (built from the spin doubling, w2s spin-support fix): full su(2) —
  each of `S_x,S_y,S_z` Hermitian, traceless, `S_alpha² = I`, and `[S_x, S_y] = 2i S_z`.
- **`S_x` is nonzero with `[H, S_x] ≠ 0`** — the transverse-spin precession source for #6
  (asserting `[H,S_x]=0` would be the physics error).
